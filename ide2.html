Sublime Text

Home Contacts Komodo VS code

In 2007 its creator left his job at Google to pursue one of his dreams: to create a better text editor.


He gave himself three principles for this software:


Discreet, minimum interface: we must be able to focus on the text and not a myriad of toolbars;


Do not hide the text by windows;

Use as much space as possible: full screen, multiscreen, side-by-side file editing should be possible.



Sublime Text integrates most of the basic features of a text editor

including customizable syntax highlighting, auto-completion, a plug-in system, etc.
The editor offers more advanced functions, however, including 4 in particular:


Minimap: preview of the whole file in a sidebar;

Selection and editing in several sections of code in parallel;

Bookmark within files;

Automatic backup;

Search and replace by regular expressions;

Support for macros and plugins in Python;

Customization of keyboard shortcuts.

The software also offers to import packages Version 3 of the software
directly offers to install PackageControl to add certain themes manually.
