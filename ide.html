There are a lot of different text editors, I will present three of them.

Komodo text editor review

Komodo Edit is a free software text editor and development environment, based on Scintilla for the editor and Firefox for rendering. It was launched in January 2007 to complement the commercial product Komodo IDE from software publisher ActiveState.

Sublime Text editor review

Sublime Text is a generic text editor coded in C ++ and Python, available on Windows, Mac and Linux. The software was designed primarily as a feature rich extension for Vim.
It was launched in January 2008

Visual Studio code editor review

Visual Studio Code is an extensible code editor developed by Microsoft for Windows, Linux, and macOS2.
It was launched in 2015
